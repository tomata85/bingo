# bingo

This is a single page application of a bingo game. The page shows a logo, a title and 5*5 matrix, each showing a short sentence. The title and the sentences are stored in a .json file.
When the user clicks an item in the matrix, there is an animation of the item being checked and a green checkmark appears on the item, and it's no longer clickable.
The checkmarks are stored in a cookie and are loaded from the cookie on the next user visit.

## Design and layout
The logo is a simple minimalistic design of the word 'Bingo'.
The design is simple, clean and minimalistic.

## Development
Simplicity, maintainability and readability of code are proriotized over modern technologies. 
Bootstrap over vanilla JS.

## Deployment
Deployment to Github Pages with a Github Action.
